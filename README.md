# 鑑定カード管理システム (Graded Card Management System)

個人開発のカード管理Webアプリケーションです。トレーディングカードの資産管理を効率化することを目的としています。

## 主な機能
- カード情報の登録、編集、削除（鑑定済みカード対応）
- 検索機能（カード名による絞り込み）
- 資産価値の自動集計（保有枚数および推定価格の合計）
- 外部マーケットサイトへのリンク（Magi、メルカリ、SNKRDUNKで相場を確認可能）
- 選択したカードの一括削除

## 技術スタック
- Backend: Java 17, Spring Boot 3
- Database: MySQL 8.0
- Frontend: Thymeleaf, Bootstrap 5
- Build Tool: Maven

## 開発の工夫点
- 検索時のユーザビリティ向上: 検索キーワード入力時にインラインでクリアボタンを表示し、ワンクリックで全件表示に戻せるように設計しました。
- UIの統一感: 画面の固定ヘッダーと統計バーをレイアウトに組み込み、データ量が増えても操作性が損なわれないようにテーブルエリアをスクロール可能な仕様にしています。
- DB移行の経験: 開発初期のH2 Databaseから、本番運用を想定したMySQLへ移行し、環境に応じた設定変更を行いました。
